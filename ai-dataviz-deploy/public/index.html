❓ 使用帮助

📤 如何上传数据？

1. 点击顶部导航的"数据上传"
2. 拖拽或点击选择文件
3. 支持 CSV、Excel、JSON 格式
4. 上传后可以预览和编辑数据

📊 如何创建图表？

1. 先上传或选择一个数据集
2. 进入"可视化编辑器"
3. 选择图表类型
4. 配置X轴、Y轴字段
5. 点击保存按钮保存图表

🔗 如何分享图表？

1. 在编辑器中创建图表
2. 点击"分享"按钮
3. 系统会生成一个分享链接
4. 复制链接发送给任何人即可

📥 如何导出图表？

1. 进入"导出中心"
2. 选择导出格式（PNG/SVG/PDF等）
3. 设置分辨率和尺寸
4. 点击确认导出
